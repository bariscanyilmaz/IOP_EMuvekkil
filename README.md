# IOP_EMuvekkil

Client management for law office.

IOP_EMuvekkil allows law suits to manage clients, arrange meetings, send messages and get reports.

## Tehcnologies 

* Dotnet Core for backend api
* Angular for client app 
* SqlServer for Database
